# ASU Eats - Smart Campus Dining

Order ahead from your favorite ASU campus restaurants with AI-powered wait time predictions. Skip the line with ASU Eats.

## Project Info

This project helps students and staff at ASU save time by predicting wait times and allowing order-ahead functionality for campus dining options.

## Getting Started

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd asu-fastpass
   ```

3. Install dependencies:
   ```sh
   npm i
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```
